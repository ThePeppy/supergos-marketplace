# Supergos Marketplace

This is the Claude Code plugin marketplace for [Supergos](https://github.com/ThePeppy/supergos).

> Based on [superpowers](https://github.com/obra/superpowers) by obra

## Installation

In Claude Code:

```bash
# Register this marketplace
/plugin marketplace add ThePeppy/supergos-marketplace

# Install the plugin
/plugin install supergos@supergos-marketplace
```

## Available Plugins

- **supergos** - Core skills library for Claude Code: TDD, debugging, collaboration patterns, and proven techniques
